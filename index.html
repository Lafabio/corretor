📋 Como configurar — Passo a passo
1. Abra Google Sheets e crie uma planilha nova. Anote o nome que der.
2. No menu superior: Extensões → Apps Script
3. Apague o código existente e cole todo o conteúdo do arquivo Code.gs que você recebeu.
4. Clique em ▶ Executar → setupInicial (só na primeira vez). Autorize as permissões.
5. Clique em Implantar → Nova implantação → Aplicativo da Web
6. Em "Quem tem acesso": selecione Qualquer pessoa. Clique em Implantar.
7. Copie a URL gerada (começa com https://script.google.com/macros/s/...) e cole no campo acima.
💾 Dados locais

Os dados ficam salvos no dispositivo mesmo sem internet. A sincronização com o Google Sheets acontece automaticamente.
